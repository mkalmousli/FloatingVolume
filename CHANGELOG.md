# Changelog

## 0.0.4 - 2026-04-21

- Added a slider size control in the app and applied it live to the floating overlay.
- Switched overlay sizing to density-independent units so it stays usable on higher-density screens.
- Refreshed the main screens with a more modern Material 3 layout and cleaner settings cards.
- Added a Gradle fallback to compile Kotlin in-process for more reliable builds in constrained environments.
